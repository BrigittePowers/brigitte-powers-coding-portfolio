Calming Colors Weather App

  • HTML/CSS
  • JavaScript
  • Mobile-First Design
  • Open Weather API

The goal of this project was to create a working weather application that can pull weather data from openweather API. The project is designed to be used on mobile. The tracker features a 5-day forecast for the searched city. The forecast includes the weekday, weather, high and low temperature, humidity, wind conditions, and UVI index; it also displays a colorized warning for extreme UVI risks.

Coding Quiz

  • JavaScript
  • Dynamic Elements
  • Local Storage
  • Timed Events

The purpose of this project was to create a functional quiz page with dynamic elements while keeping it friendly for mobile and tablet users. The quiz opens with a landing page then clears the contents in order to dynamically generate quiz questions/answers pulled from an array of objects. Scores are calculated at the end of each quiz attempt and are available for local storage to be viewed under Highscores.
